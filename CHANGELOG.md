# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] — 2025-03-31

### ✨ Added

- Initial release
- Multi-dataset browsing with automatic dataset discovery
- Document filtering by type, search query, and draft status
- Bulk selection and deletion with atomic transactions
- Paginated document table (configurable page size)
- Inline status messages for operations
- Full configuration via plugin options (`title`, `icon`, `pageSize`, `apiVersion`, `excludeTypes`, `defaultDataset`)
- TypeScript types exported for `DocumentManagerPluginConfig` and `DocEntry`
